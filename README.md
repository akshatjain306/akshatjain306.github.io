## Features

- ⚡ **Boot Sequence** — Terminal-style startup animation
- 🔮 **Particle Network** — Interactive canvas background with node connections
- ✨ **Glitch Effect** — CSS-only name glitch animation
- 🖱️ **Custom Cursor** — Neon-cyan tracking cursor with trail
- 📟 **Neofetch About** — System info styled as Linux `neofetch` output
- 🎯 **Scroll Reveal** — Intersection Observer driven section animations
- 📱 **Fully Responsive** — Mobile hamburger menu with fullscreen overlay
- 🚀 **Auto Deploy** — GitHub Actions CI/CD on every push

## Tech Stack

| Layer | Technology |
